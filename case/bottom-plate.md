# Wakizashi ボトムプレート設計仕様

## 1. 概要

ケース底面を構成するプレート。バッテリー位置決め用の浅いポケットとケーブルチャネルを内面に持つ。M2ネジでサイドケースに締結する。

| 項目 | 値 |
|---|---|
| CAD | Shapr3D |
| 素材 | PLA（3Dプリント） |
| プリンタ | Bambu Lab A1 mini（180×180×180mm） |

### 座標系

**ボトムプレート左上角を原点 (0, 0)** とする。PCB相対座標からの変換は **+2.5mm**（X, Y共通）。

> トッププレートはPCBと同一外形（204×52mm）でサイドケースに差し込む方式のため、座標系が異なる。ボトムプレートはサイドケースと同一外形（209×57mm）。

---

## 2. 外形

| 項目 | 値 |
|---|---|
| 幅 (X) | 209.000 |
| 高さ (Y) | 57.000 |
| 厚さ (Z) | 2.500 |
| 角丸半径 | R2.000（四隅すべて） |

サイドケースと同一外形。カットアウトなし。

---

## 3. バッテリーポケット（内面）

102050 LiPoバッテリー（50×20×10mm）の位置決め用。バッテリーの全高10mmはサイドケース内部空間に収まるため、ポケットは位置決め用の浅い凹みのみ。

| 項目 | 値 |
|---|---|
| ポケット形状 | 角丸矩形（R2） |
| ポケット幅 (X) | 52.000（バッテリー50mm + 片側1mmクリアランス） |
| ポケット奥行 (Y) | 22.000（バッテリー20mm + 片側1mmクリアランス） |
| ポケット深さ (Z) | 1.500（内面から掘り込み。残り板厚1.0mm） |
| 中心座標 | X = 162.500, Y = 28.500 |
| 左辺 | X = 136.500 |
| 右辺 | X = 188.500 |
| 上辺 | Y = 17.500 |
| 下辺 | Y = 39.500 |
| 根拠 | PCB相対: 中心(160, 26) → プレート: (162.5, 28.5) |

## 4. ケーブルチャネル（内面）

バッテリーケーブルをポケットから上方向に導く浅い溝。ポケット右上から上方向に伸び、ポケットと一体化した形状。プレート端までは貫通しない。

| 項目 | 値 |
|---|---|
| 起点 | ポケット右上角（X=188.5, Y=17.5） |
| 終点 | X=188.5, Y=7.5 |
| チャネル幅 | 4.000 |
| チャネル深さ | 1.500（ポケットと同じ深さ） |
| X範囲 | X=184.5〜188.5 |
| Y範囲 | Y=7.5〜17.5（ポケット上辺から上方向に10mm） |

---

## 5. M2ネジ穴（8箇所）

ボトムプレートをサイドケースにM2ネジ + ヒートインサートで締結する。ネジは底面から上向きに挿入し、サイドケース壁底部のヒートインサートにねじ込む。

### ヒートインサート仕様

| 項目 | 値 |
|---|---|
| ネジ | M2 |
| ヒートインサート | M2 × 3mm |
| インサート外径 | 約 3.0mm |
| インサート設置先 | サイドケース壁の底面 |
| ボトムプレート穴径 | φ2.400（M2ネジ通し穴） |
| 穴形状 | 貫通穴 |

### ネジ穴座標（プレート座標）

サイドケース壁の上に位置するよう配置。4隅 + 長辺各2箇所 = 8箇所。

| # | 位置 | X | Y | 備考 |
|---|---|---|---|---|
| 1 | 左上コーナー | 3.000 | 3.000 | |
| 2 | 上辺・左寄り | 71.000 | 3.000 | |
| 3 | 上辺・右寄り | 139.000 | 3.000 | |
| 4 | 右上コーナー | 206.000 | 3.000 | |
| 5 | 左下コーナー | 3.000 | 54.000 | |
| 6 | 下辺・左寄り | 71.000 | 54.000 | |
| 7 | 下辺・右寄り | 139.000 | 54.000 | |
| 8 | 右下コーナー | 206.000 | 54.000 | |

```
  ネジ穴配置図

  [1]──────────[2]──────────────[3]──────────[4]  Y=3
   │                                          │
   │                                          │
   │                                          │
  [5]──────────[6]──────────────[7]──────────[8]  Y=54
```

全ネジ穴のYを3 / 54に統一。穴端からプレート端まで最低1.8mm（3 - 1.2）の材料が確保される。

> **注意**: 印刷分割位置（X=104.5）にネジ穴がかからないよう配置済み。

## 6. 印刷分割

| 項目 | 値 |
|---|---|
| 分割位置 | X = 104.500 |
| 左半分 | 0〜104.5mm（104.5mm × 57mm） |
| 右半分 | 104.5〜209mm（104.5mm × 57mm） |
| 分割方法 | Shapr3Dで1体設計 → Bambu Studioのカット機能で分割 |
| 接合 | Bambu Studioの位置合わせピン（自動）で左右を接着 |

---

## 7. Shapr3Dモデリング手順

### Step 1: 基本外形
1. 209 × 57mm の矩形をスケッチ
2. 四隅にR2フィレット（3D）を適用
3. 2.5mm 押し出し

### Step 2: バッテリーポケット
1. 内面（上面）に52 × 22mmの矩形をスケッチ
2. 中心を (162.5, 28.5) に配置
3. 四隅にR2フィレット
4. 1.5mm 掘り込み（内面から下方向へ）

### Step 3: ケーブルチャネル
1. 内面に4 × 10mmの矩形をスケッチ（X=184.5〜188.5, Y=7.5〜17.5）
2. ポケットと同じ深さ1.5mmで掘り込み
3. ポケット右上角と連結していることを確認

### Step 4: M2ネジ穴
1. 底面（外面）に8箇所の円をスケッチ（φ2.4mm）
2. 上記座標表に従って配置
3. 貫通穴で抜く

### Step 5: 検算
- ポケット底の残り板厚: 2.5 - 1.5 = 1.0mm → 強度OK
- チャネル上端（Y=7.5）〜プレート上端（Y=0）: 7.5mm → 貫通しない
- ネジ穴がポケット・チャネルと干渉しないこと → 最寄りのネジ穴#3(139,3)はポケット上辺(Y=17.5)から14.5mm離れており干渉なし
- ネジ穴が印刷分割線（X=104.5）と重ならないこと → #2(71)と#3(139)の間で安全
