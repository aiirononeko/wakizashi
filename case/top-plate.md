# Wakizashi トッププレート設計仕様

## 1. 概要

ケース上面を構成するプレート。サイドケース内側のレッジ（段差）に載せて使用する。30箇所のスイッチカットアウトと、トップ面実装部品（XIAO・JSTコネクタ・スライドスイッチ）の逃げ穴を設ける。

| 項目 | 値 |
|---|---|
| CAD | Shapr3D |
| 素材 | PLA（3Dプリント） |
| プリンタ | Bambu Lab A1 mini（180×180×180mm） |

### 座標系

すべての座標は**PCB左上角を原点 (0, 0)** とする（PCB相対座標）。

---

## 2. 外形

| 項目 | 値 |
|---|---|
| 幅 (X) | 204.000 |
| 高さ (Y) | 52.000 |
| 厚さ (Z) | 1.300 |
| 角丸半径 | R2.000（底辺の左右2箇所のみ。上辺の左右はカットアウトで除去される） |

> トッププレートはPCBと同一外形。サイドケース内側のレッジに載せて固定する。プレート表面はサイドケース壁上端と面一になる。

## 4. XIAO逃げカットアウト（左上コーナー）

XIAOモジュール（17.78 × 21.00mm、高さ7-8mm）がトップ面左上に実装されているため、左上コーナーを矩形に切り欠く。USB-Cポートもこのカットアウトでカバーされる。

| 項目 | 値 |
|---|---|
| 形状 | 左上コーナーの矩形切り欠き |
| 右辺 | X = 21.000 |
| 下辺 | Y = 24.000 |
| 内側コーナー | R1.000（応力緩和） |
| 根拠 | XIAO右端 19.558 + 1.5mmクリアランス ≈ 21mm |
| 根拠 | XIAO下端 22.440 + 1.5mmクリアランス ≈ 24mm |

> **注意**: XIAO逃げ右辺（X=21）とC1カットアウト左辺（X=23）の間は**2mm**。狭いがスイッチクリップがプレートをPCBに固定するため構造的に問題なし。

## 7. リセットスイッチ（TL3305A）

SSOT記載の通り、外付けリセットスイッチは次リビジョンで削除予定。XIAOモジュール自体のリセットボタンがXIAO逃げカットアウトから露出するため、トッププレートでの対応は**不要**。

## 10. Shapr3Dモデリング手順

### Step 1: 基本外形
1. 204 × 52mm の矩形をスケッチ
2. 1.3mm 押し出し

### Step 2: スイッチカットアウト
1. 上面に14 × 14mmの矩形をスケッチ
2. 最初のスイッチ中心 (30, 10) に配置
3. X方向に16mmピッチで10個コピー
4. Y方向に16mmピッチで3行コピー（計30個）
5. 全カットアウトを貫通カット

### Step 3: XIAO逃げ（左上コーナー）
1. 左上から21 × 24mmの矩形をスケッチ
2. 貫通カット
3. 内側コーナー（X=21, Y=24）のエッジにR1フィレット（3D）

### Step 4: JSTコネクタ逃げ（上辺ノッチ）
1. 上辺にX=185〜195、深さ12mmの矩形をスケッチ
2. 貫通カット
3. 内側コーナー2箇所のエッジにR1フィレット（3D）

### Step 5: スライドスイッチ逃げ（右上コーナー）
1. 右上から8 × 12mmの矩形をスケッチ（X=196〜204, Y=0〜12）
2. 貫通カット
3. 内側コーナー（X=196, Y=12）のエッジにR1フィレット（3D）

### Step 6: 外形の角丸フィレット（3D）
1. 底辺の左右コーナーのエッジにR2フィレット（3D）
2. 上辺はカットアウトで除去済みのためフィレット不要

### Step 7: 検算
- XIAO逃げ右辺（X=21）〜 C1カットアウト左辺（X=23）= 2mm → OK
- JSTノッチ右辺（X=195）〜 SW逃げ左辺（X=196）= 1mm → 狭いが深さ同一のため連結しても可
- 最下カットアウト下辺（Y=49）〜 プレート下辺（Y=52）= 3mm → OK
- 最右カットアウト右辺（X=181）〜 JSTノッチ左辺（X=185）= 4mm → OK
