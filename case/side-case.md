# Wakizashi サイドケース設計仕様

## 1. 概要

トッププレートとボトムプレートの間を囲む枠状パーツ。上端にトッププレート固定用のリップ（内向きの傘）、底部にボトムプレート締結用のヒートインサートボス、北壁にUSB-C充電ポートの開口を持つ。トッププレートは下面から挿入し、リップが上方向の脱落を防止。ボトムプレートをネジ止めし、スペーサーで圧縮固定する。

| 項目 | 値 |
|---|---|
| CAD | Shapr3D |
| 素材 | PLA（3Dプリント） |
| プリンタ | Bambu Lab A1 mini（180×180×180mm） |

### 座標系

**サイドケース左上角を原点 (0, 0)** とする（ボトムプレートと共通）。PCB相対座標からの変換は **+2.5mm**（X, Y共通）。Z軸は底面（ボトムプレート接触面）を Z=0 とし、上方向が正。

## 4. リップ（トッププレート固定用の傘）

壁上端の内側に設けた内向きの張り出し。トッププレート（204×52mm）はサイドケースの底面開口（205×53mm）から挿入し、リップが上方向の脱落を防止する。ボトムプレートのネジ締めにより、スペーサー → PCB → スイッチ → トッププレート → リップ の順に圧縮固定される。

| 項目 | 値 |
|---|---|
| リップ厚さ (Z) | 1.000 |
| リップ幅（壁内面からの張り出し） | 2.000 |
| リップ上面 | Z = 17.700（壁上端から1.0mm下） |
| リップ下面 | Z = 16.700 |
| リップ内側開口 (X) | 201.000 |
| リップ内側開口 (Y) | 49.000 |
| 開口の左上角 | (4.000, 4.000) |
| 開口の右下角 | (205.000, 53.000) |
| トッププレートとの重なり（片側） | 1.500 |
| トッププレート挿入口（底面開口） | 205.000 × 53.000（壁内寸） |

リップは壁内周の上端を一周する。以下の箇所ではリップを除去し、トップ面実装部品との干渉を回避する。いずれも該当箇所はトッププレートのカットアウトと重なるため、リップの支持機能に影響なし。カバーピースで上面を覆う。

### リップ除去箇所（3箇所）

| # | 対象部品 | 除去範囲（サイドケース座標） | 根拠 |

| 1 | XIAO逃げ | X=2〜25, Y=2〜28 | トッププレートXIAO逃げ(PCB X=0〜21, Y=0〜24) + 1mmクリアランス |

> **注意**: #2と#3はX方向で隣接（X=197〜199が重複）するため、一体のカットとして処理してよい（X=186〜207, Y=2〜16）。

### カバーピース（3箇所）

トッププレートの各カットアウトにはめ込む別パーツ。XIAO/JST/スライドスイッチを上面から隠し、取り外し可能とする。

| # | 対象 | 概要 |
|---|---|---|
| 1 | XIAO逃げカバー | XIAO逃げカットアウトにはめ込み。USB-Cケーブル用の開口を設ける |
| 2 | JSTコネクタカバー | JSTノッチにはめ込み |
| 3 | SWカバー | SW逃げにはめ込み。スイッチ操作用のスロットを設ける |

> **注意**: カバーピースの詳細設計（寸法・嵌合方式・厚み）はPCB・XIAOモジュール到着後に実測して決定する。

### 固定の仕組み

- **上方向の固定**: リップがトッププレートの縁を上から押さえ、脱落を防止
- **下方向の固定**: ボトムプレート（ネジ止め）→ スペーサー → PCB → スイッチ → トッププレートの圧縮力で保持
- **横方向の位置決め**: 壁内面（205×53mm）がガイド。片側0.5mmクリアランス

## 6. スライドスイッチアクセス

MSK-12C02スライドスイッチはPCB右上コーナーに実装（PCB座標 X=201.658, Y=6.391）。トッププレートの右上コーナー切り欠き（PCB座標 X=196〜204, Y=0〜12）により上面から露出する。

サイドケース壁には専用の開口を**設けない**（初版）。トッププレート切り欠きからのアクセスで運用し、不便であれば次版で北壁または右壁にスロットを追加する。

> **根拠**: スイッチの操作頻度は低い（電源ON/OFF）。トッププレート切り欠きから指先またはピンセットで操作可能。壁の開口を増やすと剛性が下がるため、必要最小限とする。

---

## 7. リセットボタンアクセス

XIAOモジュール自体のリセットボタンは、トッププレートのXIAO逃げカットアウト（PCB座標 X=0〜21, Y=0〜24）から上面アクセス可能。サイドケースに追加の開口は**不要**。

## 9. スペーサー

PCBをトッププレート方向に押し上げ、リップとの間で圧縮固定するための部品。ボトムプレート内面とPCB底面の間に配置する。

| 項目 | 値 |
|---|---|
| 位置 | ボトムプレート内面（Z=0）〜 PCB底面付近（Z≈10.5） |
| 高さ | 約10.5mm（バッテリー空間と同等） |
| 素材 | 未定（3Dプリントスタンドオフ、シリコンパッド、またはフォーム） |
| 配置 | バッテリーを避けた位置（詳細はパーツ到着後に決定） |

> **注意**: スペーサーの具体的な形状・配置は、PCB・バッテリー・XIAOモジュール到着後に実測して決定する。高さが正確でないと圧縮不足または過圧縮になるため、実物合わせが必要。

---

## 10. 組み立て手順

1. サイドケースを裏返す（リップが下になる）
2. トッププレートをリップの上に載せる（リップが受け止める）
3. スイッチをトッププレートのカットアウトに挿し、PCBをセットする
4. スペーサーをPCB裏面側に配置する
5. ボトムプレートを重ねる
6. M2ネジ8本でボトムプレートをサイドケースに締結する
7. 全体をひっくり返して完成

```
  組み立て断面図（サイドケースを裏返した状態）

  ─────────────────────────────── ← 作業台
  ┌──┐                     ┌──┐
  │  ├─────────────────────┤  │ ← リップ（下向き）
  │  │  トッププレート       │  │ ← 1. 載せる
  │  │  PCB + スイッチ       │  │ ← 2. セットする
  │  │  スペーサー           │  │ ← 3. 置く
  │  │                     │  │
  └──┘                     └──┘
  ┌───────────────────────────┐
  │    ボトムプレート          │ ← 4. 重ねてネジ止め
  └───────────────────────────┘
```

## 12. Shapr3Dモデリング手順

### Step 1: 外殻
1. 209 × 57mm の矩形をスケッチ
2. 18.7mm 押し出し
3. 四隅の外側エッジにR2フィレット（3D）

### Step 2: 中空化
1. 底面に205 × 53mmの矩形をスケッチ（各辺から2mm内側）
2. 上方向に16.7mm カット → 壁（2mm厚）と天井（2.0mm厚）が残る

### Step 3a: ベゼル開口（壁上部）
1. 上面に205 × 53mmの矩形をスケッチ（壁内寸と一致）
2. 下方向に1.0mm カット → 壁上部に1mmのベゼル空間ができ、天井が1.0mmに薄まる

### Step 3b: リップ開口
1. Step 3aで露出した面（Z=17.7）に201 × 49mmの矩形をスケッチ（中心合わせ）
2. 下方向に1.0mm カット → 天井に開口ができ、リップ（幅2.0mmの枠）が残る

### Step 3c: リップ除去（3箇所）
1. Z=17.7の面でリップ上に以下の矩形をスケッチし、それぞれ下方向に1.0mmカット（リップ貫通）
   - **XIAO逃げ**: X=2〜25, Y=2〜28（左上コーナー）
   - **JST+SW逃げ**: X=186〜207, Y=2〜16（右上、JSTとSWを一体でカット）
2. USB-Cノッチ（X=7.2〜19.2）はStep 4で別途カットするため、XIAO逃げと一部重複してもOK

### Step 4: USB-Cノッチ
1. 北壁外面（Y=0）にノッチ形状をスケッチ
   - X: 7.2 〜 19.2（幅12mm）
   - Z: 12.7 〜 18.7（壁上端まで開放）
2. 壁厚2mm分を貫通カット
3. ノッチ下端コーナー2箇所にR1フィレット（3D）

### Step 5: ヒートインサートボス
1. 壁内側の底面（Z=0）に8箇所の円をスケッチ（φ6mm）
2. 上記座標表に従って配置
3. 上方向に4mm押し出し（ソリッド追加）
4. 各ボス上面（Z=4）中心にφ3.0mmの円をスケッチ
5. 下方向に4mm掘り込み（ボス底面Z=0まで貫通、インサート挿入口）

### Step 6: トッププレート位置合わせ確認
1. トッププレートのモデルをインポート
2. X=+2.5, Y=+2.5, Z=+15.4 でオフセット配置
3. トッププレート上面（Z=16.7）がリップ下面と一致することを確認
4. カットアウトの位置がリップ除去箇所と連続していることを確認

### Step 7: 検算
- リップ内側開口 201 × 49mm → トッププレート（204×52）は通過不可、片側1.5mm重なり → 脱落防止OK
- トッププレート挿入口 205 × 53mm → トッププレート（204×52）が片側0.5mmクリアランスで挿入可能
- リップ除去#1（X=2〜25, Y=2〜28）→ XIAO(X=4.4〜21.9, Y=3.9〜24.9)に対し全方向1mm以上のクリアランス → 干渉なし
- リップ除去#2（X=186〜207, Y=2〜16）→ JST/SW部品に対し1mm以上 → 干渉なし
- リップ南壁・西壁下部・東壁下部は除去なし → トッププレート支持は十分
- ボス上面Z=4 → PCB底面Z=10.5: 垂直クリアランス6.5mm → 干渉なし
- USB-Cノッチ（X=7.2〜19.2）→ XIAO逃げ除去範囲内 → 整合
- リップ内側開口端〜最寄りスイッチカットアウト端: Y方向1.5mm → スイッチ本体と干渉なし
- 中間ボス(X=71,139)は印刷分割線(X=104.5)から33mm以上離れている → 干渉なし
- 各半分 104.5×57×18.7mm → A1 miniビルドプレート（180×180mm）に収まる
