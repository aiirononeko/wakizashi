# Wakizashi サイドケース設計仕様

## 1. 概要

トッププレートとボトムプレートの間を囲む枠状パーツ。上端にトッププレート受けのレッジ、底部にボトムプレート締結用のヒートインサートボス、北壁にUSB-C充電ポートの開口を持つ。

| 項目 | 値 |
|---|---|
| CAD | Shapr3D |
| 素材 | PLA（3Dプリント） |
| プリンタ | Bambu Lab A1 mini（180×180×180mm） |

### 座標系

**サイドケース左上角を原点 (0, 0)** とする（ボトムプレートと共通）。PCB相対座標からの変換は **+5.0mm**（X, Y共通）。Z軸は底面（ボトムプレート接触面）を Z=0 とし、上方向が正。

---

## 4. レッジ（トッププレート受け）

壁内側の上端に設けた段差。トッププレート（204×52mm）がここに載り、表面がサイドケース壁上端と面一になる。

| 項目 | 値 |
|---|---|
| レッジ幅（片側） | 2.750 |
| レッジ深さ (Z) | 1.300（トッププレート厚と一致） |
| レッジ上面 | Z = 15.400 |
| レッジ下面（壁上端） | Z = 16.700 |
| トッププレート開口 (X) | 204.500（片側0.25mmクリアランス） |
| トッププレート開口 (Y) | 52.500（片側0.25mmクリアランス） |
| 開口の左上角 | (4.750, 4.750) |
| 開口の右下角 | (209.250, 57.250) |

レッジは壁内周を一周する。USB-C開口部ではレッジが途切れるが、該当箇所はトッププレートのXIAO逃げカットアウトと重なるため支持機能に影響なし。

## 5. USB-C開口（北壁）

XIAOモジュールのUSB-Cポート（充電用）にケーブルを接続するための開口。北壁（Y=0面）の左寄りに、上端から切り欠くノッチ形状。

| 項目 | 値 |
|---|---|
| 形状 | 上端開放ノッチ |
| 壁面 | 北壁（Y=0面） |
| 中心X | 15.700（PCB座標 10.668 + 5.0） |
| 左辺X | 9.700 |
| 右辺X | 21.700 |
| 幅 | 12.000 |
| ノッチ下端Z | 12.700 |
| ノッチ上端Z | 16.700（壁上端、開放） |
| ノッチ深さ | 4.000 |
| 内側角丸 | R1.000（下端の左右コーナー） |

> **注意**: ノッチ下端Z=12.7は推定値。PCBとXIAOモジュール到着後に実測し、必要に応じて調整する。コネクタの実際の高さによっては下端を下げる可能性がある。

## 6. スライドスイッチアクセス

MSK-12C02スライドスイッチはPCB右上コーナーに実装（PCB座標 X=201.658, Y=6.391）。トッププレートの右上コーナー切り欠き（PCB座標 X=196〜204, Y=0〜12）により上面から露出する。

サイドケース壁には専用の開口を**設けない**（初版）。トッププレート切り欠きからのアクセスで運用し、不便であれば次版で北壁または右壁にスロットを追加する。

> **根拠**: スイッチの操作頻度は低い（電源ON/OFF）。トッププレート切り欠きから指先またはピンセットで操作可能。壁の開口を増やすと剛性が下がるため、必要最小限とする。

---

## 7. リセットボタンアクセス

XIAOモジュール自体のリセットボタンは、トッププレートのXIAO逃げカットアウト（PCB座標 X=0〜21, Y=0〜24）から上面アクセス可能。サイドケースに追加の開口は**不要**。

---

## 8. ヒートインサートボス（8箇所）

壁内側の底部に設ける円筒形ボス。M2ヒートインサートを埋め込み、ボトムプレートからのM2ネジを受ける。

### ボス仕様

| 項目 | 値 |
|---|---|
| ボス外径 | φ6.000 |
| インサート下穴径 | φ3.200（M2ヒートインサート用、要実測調整） |
| ボス高さ | 4.000（インサート長と一致） |
| ボス底面 | Z = 0.000（壁底面と面一、底面に開口） |
| ボス上面 | Z = 4.000（閉塞） |
| PCBまでの垂直クリアランス | 7.600（11.6 − 4.0） |

### ボス座標（サイドケース座標 = ボトムプレート座標）

| # | X | Y | 位置 | 壁との関係 |
|---|---|---|---|---|
| 1 | 3.000 | 3.000 | 左上コーナー | 2壁に接続 |
| 2 | 72.000 | 3.000 | 上辺・左寄り | 北壁に接続 |
| 3 | 142.000 | 3.000 | 上辺・右寄り | 北壁に接続 |
| 4 | 211.000 | 3.000 | 右上コーナー | 2壁に接続 |
| 5 | 3.000 | 59.000 | 左下コーナー | 2壁に接続 |
| 6 | 72.000 | 59.000 | 下辺・左寄り | 南壁に接続 |
| 7 | 142.000 | 59.000 | 下辺・右寄り | 南壁に接続 |
| 8 | 211.000 | 59.000 | 右下コーナー | 2壁に接続 |

```
  ボス配置図（底面から見上げた図）

  [1]──────────[2]──────────────[3]──────────[4]  Y=3
   │                                          │
   │                                          │
   │                                          │
  [5]──────────[6]──────────────[7]──────────[8]  Y=59
```

コーナーのボス（#1, #4, #5, #8）は隣接する2壁から支持される。中間のボス（#2, #3, #6, #7）は1壁から内側に突出する。いずれもPCB領域（X=5〜209, Y=5〜57）の外側に位置し、水平方向の干渉なし。

## 9. 印刷分割

| 項目 | 値 |
|---|---|
| 分割位置 | X = 107.000 |
| 左半分 | 0〜107mm（107 × 62 × 16.7mm） |
| 右半分 | 107〜214mm（107 × 62 × 16.7mm） |
| 分割方法 | Shapr3Dで1体設計 → Bambu Studioのカット機能で分割 |
| 接合 | Bambu Studioの位置合わせピン（自動）で左右を接着 |
| 印刷向き | 壁上端（レッジ面）を下にしてビルドプレートに接地 → レッジ面が平滑に仕上がる |

## 10. Shapr3Dモデリング手順

### Step 1: 外殻
1. 214 × 62mm の矩形をスケッチ
2. 16.7mm 押し出し
3. 四隅の外側エッジにR2フィレット（3D）

### Step 2: 中空化
1. 上面に210 × 58mmの矩形をスケッチ（各辺から2mm内側）
2. 下方向に16.7mm 貫通カット → 枠形状になる

### Step 3: レッジ
1. 壁内側の上面（Z=16.7）に矩形リングをスケッチ
   - 外形: 210 × 58mm（壁内面と一致）
   - 内形: 204.5 × 52.5mm
2. 下方向に1.3mm押し出し（ソリッド追加）
3. レッジ上面が Z=15.4 になることを確認

### Step 4: USB-Cノッチ
1. 北壁外面（Y=0）にノッチ形状をスケッチ
   - X: 9.7 〜 21.7（幅12mm）
   - Z: 12.7 〜 16.7（壁上端まで開放）
2. 壁厚2mm分を貫通カット
3. ノッチ下端コーナー2箇所にR1フィレット（3D）

### Step 5: ヒートインサートボス
1. 壁内側の底面（Z=0）に8箇所の円をスケッチ（φ6mm）
2. 上記座標表に従って配置
3. 上方向に4mm押し出し（ソリッド追加）
4. 各ボス上面（Z=4）中心にφ3.2mmの円をスケッチ
5. 下方向に4mm掘り込み（ボス底面Z=0まで貫通、インサート挿入口）

### Step 6: 検算
- レッジ開口 204.5 × 52.5mm → トッププレート（204×52）が片側0.25mmクリアランスで収まる
- ボス上面Z=4 → PCB底面Z=11.6: 垂直クリアランス7.6mm → 干渉なし
- USB-Cノッチ（X=9.7〜21.7）→ トッププレートXIAO逃げ（PCB X=0〜21 = サイドケース X=5〜26）の範囲内 → レッジ途切れの影響なし
- ボス#3(142,3) → ポケット/チャネルとはZ方向で完全に分離（ボスはサイドケース内、ポケットはボトムプレート内面）
- 中間ボス(X=72,142)は印刷分割線(X=107)から35mm以上離れている → 干渉なし
- 各半分 107×62×16.7mm → A1 miniビルドプレート（180×180mm）に収まる
