Wakizashi Firmware Flasher

ブラウザから直接 Wakizashi のファームウェアを書き込みます。Chrome / Edge / Opera の最新版で動作します。

ブラウザ互換性

  • WebHID: …
  • WebSerial: …

1 ファームウェアを取得

このページに同梱された最新の DFU パッケージを読み込みます。 新しい Release が公開されると自動で更新されます。

—
ローカルファイルを使う

手元の wakizashi-dfu.zip を選択すると、それを使って書き込みます。

2 キーボードをブートローダへ

USB ケーブルで PC に接続した状態で、下のボタンを押してキーボードを選択してください。自動で DFU モードに入ります。

—

ボタンが効かない場合は、キーボードのリセットボタンを素早く 2 回押して手動で DFU モードに入ってください。

3 書き込み

ブートローダが列挙されたあと、下のボタンでシリアルポート (通常 Adafruit または Seeed で始まる名前) を選んでください。
